WHY

Created to help find and embed .woff fonts into the initial 14kb window

BUILT WITH

  • Google Fonts: GitHub repository, website
  • Key NPM Packages: fontmin, fontkit, Fonts Similarizer
  • DataTables jQuery plugin
  • Gomix - Hosts an internally used font subset generation API

THE CODE

Some of it's a mess, maybe I'll clean it up / refactor if this thing gets some interest.

  • Inlined Woffs - Sources files for a Metalsmith pipeline that generates this static site
  • Inline Woff Finder - Script to identify potentially well woff inlined fonts
  • TTF to WOFF subset - GoMix hosted code for an API that takes a .ttf file and set of glyphs to generate a .woff
